# Ceb


## Installation

```
conda create -n Ceb python=3.12
conda activate Ceb

conda install pytorch==2.3.1 torchvision==0.18.1 torchaudio==2.3.1 pytorch-cuda=11.8 -c pytorch -c nvidia

conda install -c conda-forge scikit-image opencv pillow numpy scikit-learn cvxpy

pip install cvxopt matplotlib

pip install anytree
```
